**xyz-reader 是 Android 平台上一款阅读文章的App 。应用多处展现了MD风格的实现，阅读界面可以通过左右滑动来切换文章。我在此应用中负责以下优化工作**
- 使得Toolbar和FAB根据用户的动作进行隐藏和显示
- 在合适的地方将图片扩展至状态栏
- 根据google的材料设计规范为View设置合适的 Elevation
- 根据google的材料设计规范为应用设置合适的主题颜色
- 对图片使用合适的裁剪方式
- 使用更易于阅读的字体
